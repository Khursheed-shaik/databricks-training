
# Day 4 - College Management System SQL Practice

## What I Learned

Today I practiced advanced SQL concepts using a College Management System database containing departments, staff, students, subjects, and marks tables. I explored relational database design and worked with real-world scenarios involving joins, aggregates, subqueries, NULL handling, and window functions.

I learned how to analyze academic and departmental data using SQL queries and improved my understanding of ranking, cumulative calculations, partitioning, and analytical functions.

## Topics Covered

* INNER JOIN, LEFT JOIN, and multi-table joins
* Aggregate functions with GROUP BY and HAVING
* NULL value handling
* Subqueries and correlated queries
* Ranking functions such as RANK(), DENSE_RANK(), and ROW_NUMBER()
* Analytical functions using OVER()
* LAG() and LEAD() functions
* Running totals and cumulative averages
* NTILE(), PERCENT_RANK(), and CUME_DIST()
* Department-wise and student-wise performance analysis

* Practiced joins, aggregates, filtering, ranking, and analytical queries

This practice significantly improved my SQL problem-solving skills and gave me practical exposure to advanced SQL concepts used in real database systems and data analysis workflows.
