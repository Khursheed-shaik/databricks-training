
# Day 2 - SQL Query Practice

## What I Learned

Today I practiced SQL queries using different clauses and operators on an Employees table. I learned how to filter records using conditions, perform grouping and aggregation, retrieve distinct values, and use logical, comparison, BETWEEN, IN, and LIKE operators to analyze data more effectively.

## Work Done

* Created and populated the Employees table using SQL
* Solved queries on SELECT, WHERE, GROUP BY, HAVING, DISTINCT, and TOP operations
* Practiced filtering data using logical and comparison operators
* Explored pattern matching and range-based queries using LIKE and BETWEEN
* Tested queries using DB-Fiddle and uploaded work to GitHub

## Key Takeaway

This practice improved my SQL fundamentals and helped me understand how different SQL clauses work together for efficient data retrieval and analysis.
