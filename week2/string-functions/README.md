# String Functions in SQL

## What I Learned

Today I practiced SQL string manipulation functions using an Employees table. I learned how to modify, clean, format, search, and extract text data using built-in MySQL string functions.

## Topics Covered

* LENGTH() and CHAR_LENGTH()
* UPPER() and LOWER()
* TRIM(), LTRIM(), and RTRIM()
* CONCAT() and CONCAT_WS()
* SUBSTRING() and SUBSTR()
* LEFT() and RIGHT()
* INSTR() and LOCATE()
* REPLACE()
* REVERSE()
* LPAD() and RPAD()
* IFNULL() and COALESCE()
* FIND_IN_SET()

## Work Done

* Created Employees table and inserted sample data
* Practiced multiple string manipulation queries
* Worked with email, department, city, and remarks data
* Cleaned and formatted string values using SQL functions
* Explored real-world text processing scenarios
* Tested all queries using DB-Fiddle/MySQL

## Key Takeaway

This practice improved my understanding of SQL string functions and helped me learn how text data is processed, formatted, and analyzed in real-world database applications.
