
# NULL Functions in SQL

## What I Learned

Today I practiced handling NULL values in SQL using Employees, Orders, and Products tables. I learned how NULL values affect calculations, filtering, and query results, and how SQL functions can be used to replace or manage missing data effectively.

## Topics Covered

* IS NULL and IS NOT NULL
* IFNULL() / ISNULL()
* COALESCE()
* NULLIF()
* Handling NULL values in calculations
* Real-time business scenarios using NULL handling
* Preventing divide-by-zero errors
* Conditional data replacement

## Work Done

* Created Employees, Orders, and Products tables
* Inserted sample data with NULL values
* Solved NULL handling practice questions
* Practiced replacing and filtering NULL values
* Worked on real-world business use cases involving missing data
* Tested all queries using DB-Fiddle/MySQL

## Key Takeaway

This practice improved my understanding of NULL handling in SQL and helped me learn how to safely manage missing or incomplete data in real-world database applications.
