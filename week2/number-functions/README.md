# Number Manipulation Functions in SQL

## What I Learned

Today I practiced numeric and mathematical functions in SQL using an Employee Salary table. I learned how to perform calculations, rounding operations, formatting, and mathematical manipulations using built-in MySQL numeric functions.

## Topics Covered

* ABS()
* ROUND()
* CEIL() and FLOOR()
* TRUNCATE()
* MOD()
* POWER() and POW()
* SQRT()
* SIGN()
* RAND()
* FORMAT()
* GREATEST()
* LEAST()

## Work Done

* Created Employee Salary table and inserted sample data
* Practiced numeric manipulation functions using SQL queries
* Worked with salary, bonus, tax percentage, and experience data
* Performed rounding, formatting, and mathematical calculations
* Explored real-world salary and numeric analysis scenarios
* Tested all queries using DB-Fiddle/MySQL

## Key Takeaway

This practice improved my understanding of SQL numeric functions and helped me learn how mathematical operations and data formatting are used in real-world business and financial analysis.
