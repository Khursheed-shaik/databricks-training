
# SQL Joins and Relationship Queries

## What I Learned

Today I practiced SQL JOIN operations using Employees, Departments, and Projects tables. I learned how relational tables are connected using common keys and how different JOIN types help retrieve both matching and unmatched records from multiple tables.

I also explored employee-manager relationships using self joins and understood how SQL joins are used in real-world business scenarios.

## Topics Covered

* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* FULL OUTER JOIN
* SELF JOIN
* Multi-table joins
* Employee-manager relationship queries
* Department and project mapping
* Handling NULL values in joins
* Aggregate queries with joins
