
# Date and Timestamp Functions in MySQL

## What I Learned

Today I practiced MySQL date and timestamp functions using an Orders table. I learned how to work with date-based calculations, extract date parts, perform date arithmetic, format timestamps, and analyze business-related date scenarios.

## Topics Covered

* CURDATE(), CURTIME(), NOW()
* YEAR(), MONTH(), DAY()
* EXTRACT()
* MONTHNAME() and DAYNAME()
* WEEKDAY() and DAYOFWEEK()
* DATE_ADD() and DATE_SUB()
* DATEDIFF() and TIMESTAMPDIFF()
* LAST_DAY()
* DATE_FORMAT()
* STR_TO_DATE()
* Financial year calculations using CASE

## Work Done

* Created Orders table and inserted sample data
* Practiced date extraction and formatting queries
* Worked on date arithmetic and filtering operations
* Explored real-time business use cases using SQL date functions
* Tested all queries using DB-Fiddle/MySQL

## Key Takeaway

This practice improved my understanding of handling dates and timestamps in SQL and helped me learn how date functions are used in real-world business reporting and analytics.
