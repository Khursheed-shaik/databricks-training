# Databricks Training - Week 1

## Overview

This repository contains my Week 1 SQL practice and database learning activities as part of Databricks/Data Engineering training. During this week, I practiced SQL concepts from basic queries to advanced analytical queries using multiple datasets and real-world database scenarios.

The practice work was organized day-wise to maintain consistency, track learning progress, and improve problem-solving skills in SQL.

---

# Week 1 Learning Summary

## Day 1 - SQL Fundamentals and Intermediate Queries

Worked with Employee, Department, and Project tables to practice core SQL concepts and intermediate-level queries.

### Topics Covered

* SELECT statements
* WHERE conditions
* LIKE operator
* ORDER BY
* GROUP BY and HAVING
* Aggregate functions
* INNER JOIN and LEFT JOIN
* Nested queries and subqueries
* Department-wise and employee-wise analysis

### Work Done

* Executed `datacreation.sql`
* Solved SQL queries from Q1 to Q65
* Practiced joins, filtering, sorting, grouping, and aggregate analysis
* Tested queries using DB-Fiddle

## Day 2 - SQL Operators and Data Filtering

Practiced SQL operations using an Employees table with real-world employee data.

### Topics Covered

* SELECT and WHERE queries
* GROUP BY and HAVING
* DISTINCT
* TOP queries
* Comparison operators
* Logical operators
* IN and NOT IN
* BETWEEN
* LIKE operator

### Work Done

* Created Employees table and inserted sample data

* Improved understanding of SQL conditions and operators

Focused on relational database concepts using students, instructors, courses, and enrollments tables.

### Topics Covered

* INNER JOIN
* LEFT JOIN

* FULL OUTER JOIN
* CROSS JOIN
* Handling unmatched records
* Multi-table joins

### Work Done

Worked on an advanced College Management System database with realistic academic data.

### Topics Covered
